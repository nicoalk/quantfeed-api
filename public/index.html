QuantFeed

A live financial data API, payable by AI agents through InFlow's x402 and MPP payment protocols.

QuantFeed is a real, deployed service — not a demo. It serves real-time stock prices, company overview data, and business news, and is metered per call using InFlow's agent-native payment protocols (x402 and MPP), so an AI agent can pay for and consume data autonomously, with no API keys or manual checkout required.

Endpoints

GET /stock-price?ticker=<symbol>

Real-time stock price data for the given ticker.

GET /company-overview?ticker=<symbol>

Company overview data for the given ticker.

GET /business-news

Business news.

Pricing

Endpoint Price per call
/stock-price$0.01
/company-overview$0.02
/business-news$0.02
Note: Opening any of the endpoints above directly in a browser will return a 402 Payment Required response. This is expected, correct behavior — not an error. These endpoints are metered for AI agents via InFlow's x402/MPP protocols, and payment happens as part of that protocol handshake rather than through a browser session.
